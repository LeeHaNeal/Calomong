# Calomong 🥗🏃‍♂️

## 프로젝트 소개
개인 맞춤형 건강관리 웹 서비스입니다.  
사용자가 섭취한 칼로리와 소모한 칼로리를 시각적으로 확인하고, 랭킹과 챌린지 기능을 통해 동기 부여를 제공합니다.  
React, Spring Boot, Oracle DB 기반이며, 팀장으로서 기획과 백엔드 개발을 주도했습니다.

## 주요 기능
- 회원가입/로그인, 세션 관리
- 식단 및 운동 기록, 칼로리 요약
- 챌린지 참여, 실시간 점수 반영
- 랭킹 시스템, 사용자별 맞춤 데이터 제공

## 개발 환경
- **프론트엔드:** React  
- **백엔드:** Spring Boot  
- **데이터베이스:** Oracle DB  
- **언어:** Java, SQL  
- **도구:** STS4, VS Code  
- **버전 관리:** Git, GitHub  

## 담당 역할
- 팀장: 전체 일정 관리, 기획, 기능 조율
- 백엔드: Spring Boot 서버 구축, DB 설계, 인증/세션 관리
- 챌린지/랭킹 시스템 설계 및 구현
- 기능 통합 및 최종 QA 테스트
