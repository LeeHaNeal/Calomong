## ✨ 프로젝트 소개
최근 저당 식단과 운동 루틴 관리 등 개인 맞춤형 건강관리 트렌드에 대응하여,  
사용자가 섭취한 칼로리와 소모한 칼로리를 시각적으로 파악하고,  
랭킹과 챌린지 기능을 통해 스스로 동기를 부여할 수 있도록 설계한 웹 기반 건강관리 서비스입니다.  

팀장으로서 전체 일정 관리와 기획, 백엔드 개발을 주도했습니다.

## 🔥 주요 기능
### ✅ 회원 & 인증
- 이메일 회원가입/로그인, 세션 관리
- 사용자 권한별 기능 접근 제어

### ✅ 식단 & 운동 관리
- 일자별 섭취 칼로리 기록 및 요약
- 운동 기록 등록 및 소모 칼로리 계산

### ✅ 챌린지 & 랭킹
- 챌린지 참여 시점 점수 반영
- 실시간 랭킹 확인 가능
- 도전 참여로 사용자 동기 부여

### ✅ 데이터 시각화
- 캘린더 기반 기록 확인
- 차트와 그래프를 통한 섭취/소모 데이터 시각화
